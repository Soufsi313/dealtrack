# DealTrack

DealTrack est une application Python permettant d'être notifié lorsqu'une promotion est disponible sur un jeu, une console ou du matériel informatique. L'utilisateur peut créer une liste de souhaits sous forme de mots-clés (ex : "Resident Evil 2 Remake", "PS5", "RTX 4080"). Lorsqu'une promotion correspondant à un mot-clé est détectée sur un site marchand (Amazon, Instant Gaming, etc.), une notification s'affiche sur l'ordinateur et le site de la promotion est ouvert.

## Fonctionnalités principales
- Création et gestion d'une liste de souhaits
- Détection automatique des promotions sur des sites marchands
- Notification instantanée sur l'ordinateur
- Ouverture du site marchand en cas de promotion

## Structure du projet
- DealTrack/ : Dossier principal
- notifications/ : Gestion des notifications
- wishlist/ : Gestion de la liste de souhaits
- promotions/ : Détection des promotions
